# TIP_CALCULATOR

> Calculatrice de pourboire web avec répartition par personne

Application web statique minimaliste pour calculer rapidement le pourboire au restaurant et le montant à payer par personne.

## Installation

Aucune installation requise. Ouvrir `src/index.html` dans un navigateur.

## Utilisation

1. Entrer le montant de l'addition
2. Entrer le nombre de personnes
3. Sélectionner le pourcentage de pourboire
4. Le calcul se fait en temps réel :
   - Montant du pourboire
   - Total avec pourboire
   - Montant par personne

## Déploiement

Hébergement statique recommandé :
- GitHub Pages
- Netlify
- Vercel
